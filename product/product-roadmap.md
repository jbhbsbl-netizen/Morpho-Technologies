# Roadmap

Building on: Defense tech startup patterns (Vatn Systems style), Vercel deployment, Next.js + Tailwind

## Sections

### 1. Foundation & Branding
Set up project structure, design logo (Morpho butterfly concept with navy/teal scheme), establish color palette and typography, configure Next.js + Tailwind CSS.

### 2. Website Build
Build all pages in one cohesive section — Hero (AUV photos + one-line pitch), About/Technology (cost advantage story), Applications (defense use cases), Team (3 Purdue students), Contact form.

### 3. Deploy & Polish
Deploy to Vercel, get working URL (morphotechnologies.vercel.app or custom domain), test on mobile/desktop, final polish and validation.
